# Victor Desktop App

Native Tauri shell for Victor.

```text
frontend/      Static HTML, CSS, and JavaScript UI
src-tauri/     Rust commands and Tauri configuration
```

Run from the repository root:

```bash
npm run desktop
```

The composer has a `Think` toggle. Off sends `THINK=false` for fast daily answers; on sends `THINK=true` for deeper reasoning on the next request.
